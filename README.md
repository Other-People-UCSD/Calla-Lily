# Calla-Lily
Other People Magazine's website rebuilt in React JS as a modern static site generator

Refer to the [documentation](https://docs-other-people.vercel.app/) for this repo's architecture, instructions, and additional information.

## Installation

```
$ yarn
```

## Local Development

```
$ yarn dev
```

## Linting

```
$ yarn audit
```

This command checks to make sure the Next program runs properly as well as checking that TinaCMS renders everything. 

## Build

```
$ yarn build
```

This command generates the static website content into the `.next/out` directory.
